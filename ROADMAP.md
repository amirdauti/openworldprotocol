# Roadmap (phase-based)

Milestone A — Foundations
- Monorepo scaffolding
- Protocol skeleton (handshake + versioning)
- Server boots + listens on port
- Unity connects + basic handshake
- Client boots and starts local server process (one-click host)

Milestone B — Worlds are playable
- Chunk streaming + persistence
- Basic movement/entities
- Universe map (from local list / manual connect)
- Home/Character screen + avatar preview/customization

Milestone C — Local generation
- Claude + Codex provider adapters
- Generate world manifest + chunks + assets into a workspace directory
- In-client “generate character” + “generate world”
- AI companion (“orb”) UI + provider selection + settings

Milestone D — Discovery
- On-chain registry program + client reader
- Universe map renders real worlds from registry

Milestone E — Token per world
- Private launchpad integration path
- World manifest includes mint/pool metadata
- Client displays token info + basic currency UI hooks
- Wallet adapter signing flow surfaced from client (host-only)
